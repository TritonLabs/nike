Wagmify Commands

Command Example Description
help, commands commands Lists available commands.
p, price, coin p ohm Opens CoinGecko dashboard for the coin.
tx tx 0x000.. or tx 1123123 for SOL Opens Etherscan or Solscan depending on the hash type.
w, wallet, a, address, contract w kunalm.eth Opens the walllet/address based on the hash or the ENS name.
nft nft chain runners Searches for the NFT keyword in the OpeanSea.
s, swap swap ohm dai Opens uniswap with the two coins to swap.
ipfs ipfs Qmd8nEQ.... Opens the file with the hash on IPFS.
ipns ipns kunalm.xyz Opens the domain on IPNS.
ar, arweave ar 9mUpmZPWR... Opens the file with the hash on ARWeave.

Getting Started

  1. Open Chrome and click the three dots. Click Settings and scroll down to Search engines.
  2. Click Manage search engines.
  3. Add a new search engine with the URL https://wagmify.me/?q=%s. Set both name and keyword to wagmify.
  4. Click on the tripple dot and make this the default search engine.
